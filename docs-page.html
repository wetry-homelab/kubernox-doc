Kubernox Basics

This is the basic configuration required for minimal setup of kubernox. Everything is deploy in local docker, except Proxmox informations, all is optional.

1. Create file name kubernox.yaml and paste the content from the default file (sample behind).
2. After file created, launch command
chmod a+x Kubernox
./Kubernox
